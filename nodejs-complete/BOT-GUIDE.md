## 🔧 การตั้งค่าและการใช้งานขั้นสูง

### 📱 การใช้งานในกลุ่ม
1. เพิ่มบอทเข้ากลุ่ม: https://t.me/police_ddb_bot?startgroup=start
2. ตั้งเป็น Admin (ถ้าต้องการ)
3. ใช้คำสั่งได้เหมือนการสนทนาส่วนตัว

### 🔒 ความปลอดภัย
- ข้อมูลถูกเข้ารหัสและจัดเก็บใน Google Sheets
- Rate limiting (100 requests ต่อ 15 นาที)
- การตรวจสอบประเภทไฟล์
- ป้องกันการอัปโหลดไฟล์อันตราย

### 📊 ข้อจำกัด
- **ไฟล์**: สูงสุด 50MB ต่อไฟล์
- **ไฟล์รวม**: สูงสุด 100 ไฟล์ต่อผู้ใช้
- **การค้นหา**: แสดงผล 10 รายการแรก
- **ข้อความ**: สูงสุด 4,096 ตัวอักษร (ข้อจำกัดของ Telegram)

## 🆘 การแก้ไขปัญหา

### ปัญหาที่พบบ่อย

**บอทไม่ตอบ:**
```
1. ตรวจสอบว่าบอทออนไลน์อยู่
2. ลองส่ง /start ใหม่
3. ตรวจสอบการเชื่อมต่อ internet
```

**ไม่สามารถบันทึกข้อมูลได้:**
```
1. ตรวจสอบรูปแบบข้อมูลที่ส่ง
2. ลองส่งข้อมูลให้สั้นลงและชัดเจนขึ้น
3. ตรวจสอบว่าไม่มีตัวอักษรพิเศษ
```

**การค้นหาไม่แสดงผล:**
```
1. ลองใช้คำค้นหาที่สั้นกว่า
2. ตรวจสอบการสะกดคำ
3. ลองค้นหาด้วยข้อมูลอื่น (เช่น เบอร์โทร แทนชื่อ)
```

**ไฟล์อัปโหลดไม่ได้:**
```
1. ตรวจสอบขนาดไฟล์ (สูงสุด 50MB)
2. ตรวจสอบประเภทไฟล์ที่อนุญาต
3. ลองเปลี่ยนชื่อไฟล์ (ไม่ใช้อักขระพิเศษ)
```
