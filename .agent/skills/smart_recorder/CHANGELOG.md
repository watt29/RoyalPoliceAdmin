# Changelog: Smart Recorder Skill

## [1.0.0] - 2026-02-25
### Added
- **Mashed Number Splitter**: ระบบแยกตัวเลขติดกันด้วยสมการคณิตศาสตร์ (Total = InUse + InStock)
- **Auto-Shift Detection**: ระบบตรวจจับและข้ามคอลัมน์ลำดับ (1, 2, 3) อัตโนมัติเมื่อเกิดการเขยิบของ Index
- **Multi-Category Detection**: รองรับหมวดหมู่ เสื้อเกราะ, อาวุธ, กระสุน, ยานพาหนะ
- **Markdown Header Prefix Stripping**: รองรับคำสั่ง "จด", "บันทึก" นำหน้าหัวตาราง

### Fixed
- แก้ปัญหาดึงข้อมูลผิดช่องเมื่อมีคอลัมน์ "จำนวน" ซ้ำกันในหัวตาราง
- แก้ไขตัวสะกด "ยุานพาหนะ" -> "ยานพาหนะ"
