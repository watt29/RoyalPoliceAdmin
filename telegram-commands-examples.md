# 🤖 Telegram Commands สำหรับ File Operations

## 📝 **การสร้างไฟล์ (CREATE)**

### วิธีใช้:
```bash
# รูปแบบ 1: แยกข้อความ
/สร้าง report.txt
รายงานประจำวัน 30/03/2567
- เหตุการณ์สำคัญ
- สถิติการจับกุม

# รูปแบบ 2: บรรทัดเดียว
สร้างไฟล์ notes.md: "# บันทึกสำคัญ\n- ประชุม 14:00"

# รูปแบบ 3: สร้าง CSV
/สร้าง contacts.csv
ชื่อ,ตำแหน่ง,เบอร์โทร
สมชาย,พ.ต.อ.,081-234-5678
สมหญิง,ร.ต.ต.,081-345-6789
```

## ✏️ **การแก้ไขไฟล์ (EDIT)**

### วิธีใช้:
```bash
# แก้ไขทั้งไฟล์
/แก้ไข report.txt
รายงานประจำวัน 30/03/2567 (แก้ไข)
- เหตุการณ์สำคัญ: ไม่มี
- สถิติการจับกุม: 5 คดี

# แก้ไขบรรทัดเฉพาะ (advance)
/แก้ไข notes.md บรรทัด 2: "- ประชุม 15:30 (เลื่อนเวลา)"

# อัปเดตไฟล์
อัปเดต report.txt: "เพิ่มข้อมูลใหม่..."
```

## 🗑️ **การลบไฟล์ (DELETE)**

### วิธีใช้:
```bash
# ลบไฟล์
/ลบ old_report.txt

# ลบพร้อม confirm
/ลบ important.pdf --confirm

# ลบหลายไฟล์
ลบไฟล์ temp*.txt

# ย้ายไปถังขยะ
/ขยะ report.txt
```

## 📄 **การอ่านไฟล์ (READ)**

### วิธีใช้:
```bash

/อ่าน report.txt
/ดู notes.md

# แสดงบางส่วน
/อ่าน log.txt บรรทัด 1-10

# อ่านแบบ formatted
อ่านไฟล์ data.csv แบบตาราง
```

## 📁 **การจัดการไฟล์ (MANAGEMENT)**

```bash
/รายชื่อ
```

## 🎯 **ตัวอย่างการใช้งานจริง**

### สร้างรายงานประจำวัน:
```bash
User: /สร้าง รายงาน-30-03-67.md
User: # รายงานประจำวัน 30/03/2567
## เหตุการณ์สำคัญ
- ไม่มีเหตุการณ์สำคัญ

## สถิติ
- การจับกุม: 3 คดี
- อุบัติเหตุ: 1 ครั้ง

Bot: ✅ สร้างไฟล์ รายงาน-30-03-67.md สำเร็จ (ขนาด 156 ไบต์)
```

### แก้ไขข้อมูลเร่งด่วน:
```bash
User: /แก้ไข รายงาน-30-03-67.md
User: # รายงานประจำวัน 30/03/2567
## เหตุการณ์สำคัญ
- 🚨 เกิดอุบัติเหตุร้ายแรง ถ.พหลโยธิน กม.15

Bot: ✅ แก้ไขไฟล์ รายงาน-30-03-67.md สำเร็จ
📁 สำรองไฟล์เก่า: รายงาน-30-03-67.md.backup.20240330_143022
```

### ดูสถานะไฟล์:

```bash
📊 เหลือ: 97 ไฟล์
```

## 🔒 **ความปลอดภัย**

### File Type ที่อนุญาต:
- ✅ `.txt, .md, .log` - ไฟล์ข้อความ
- ✅ `.csv, .json` - ไฟล์ข้อมูล
- ✅ `.pdf, .docx` - เอกสาร
- ❌ `.exe, .bat, .sh` - ไฟล์อันตราย

### ข้อจำกัด:
- **ขนาดไฟล์:** สูงสุด 50 MB ต่อไฟล์
- **จำนวนไฟล์:** สูงสุด 100 ไฟล์ต่อผู้ใช้
- **การสำรอง:** ไฟล์เก่าถูกสำรองก่อนแก้ไข
- **ถังขยะ:** ไฟล์ที่ลบจะเก็บไว้ในถังขยะ
